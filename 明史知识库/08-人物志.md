# 明朝人物志

## 一、名臣

### 1. 刘伯温 (1311-1375)
- **籍贯**：浙江青田
- **身份**：明朝开国功臣
- **主要事迹**：
  - 辅佐朱元璋建立明朝
  - 参与制定《大明律》
  - 预言"胡惟庸案"
- **评价**："三分天下诸葛亮，一统江山刘伯温"

### 2. 李善长 (1314-1390)
- **籍贯**：安徽定远
- **身份**：明朝开国功臣，首任丞相
- **主要事迹**：
  - 辅佐朱元璋建立明朝
  - 制定各项制度
- **结局**：因胡惟庸案被赐死

### 3. 徐达 (1332-1385)
- **籍贯**：安徽凤阳
- **身份**：明朝开国名将
- **主要事迹**：
  - 北伐灭元，攻占大都
  - 多次远征蒙古
- **评价**："明朝第一功臣"

### 4. 方孝孺 (1357-1402)
- **籍贯**：浙江宁海
- **身份**：建文帝重臣
- **主要事迹**：
  - 拒绝为朱棣起草登基诏书
  - 被诛十族（873人）
- **评价**："天下读书种子绝矣"

### 5. 杨士奇 (1365-1444)
- **籍贯**：江西泰和
- **身份**：明朝名臣，内阁首辅
- **主要事迹**：
  - 历仕四朝，辅佐仁宣之治
  - 内阁制度的奠基人
- **评价**："杨士奇廉慎，有古大臣之风"

### 6. 于谦 (1398-1457)
- **籍贯**：浙江杭州
- **身份**：明朝名臣
- **主要事迹**：
  - 北京保卫战，击退瓦剌
  - 被英宗冤杀
- **名言**："粉骨碎身浑不怕，要留清白在人间"
- **评价**："于谦之忠，可比岳飞"

### 7. 张居正 (1525-1582)
- **籍贯**：湖北江陵
- **身份**：明朝内阁首辅
- **主要事迹**：
  - 推行一条鞭法
  - 实行考成法
  - 整顿吏治
- **评价**："明代唯一的大政治家"

### 8. 海瑞 (1514-1587)
- **籍贯**：广东琼山
- **身份**：明朝清官
- **主要事迹**：
  - 抬棺上疏，批评嘉靖帝
  - 清廉自守
- **评价**："海青天"

## 二、名将

### 1. 徐达 (1332-1385)
- **主要战绩**：
  - 北伐灭元，攻占大都
  - 多次远征蒙古
- **评价**：明朝开国第一功臣

### 2. 常遇春 (1330-1369)
- **籍贯**：安徽怀远
- **主要战绩**：
  - 北伐灭元
  - 鄱阳湖之战
- **评价**："常十万"（号称率十万兵）

### 3. 戚继光 (1528-1588)
- **籍贯**：山东蓬莱
- **主要事迹**：
  - 创建戚家军
  - 发明鸳鸯阵
  - 基本平定东南沿海倭患
  - 编著《纪效新书》《练兵实纪》
- **评价**："封侯非我意，但愿海波平"

### 4. 李成梁 (1526-1615)
- **籍贯**：辽宁铁岭
- **主要事迹**：
  - 镇守辽东30年
  - 多次击败蒙古、女真
- **评价**：辽东边防名将

### 5. 袁崇焕 (1584-1630)
- **籍贯**：广东东莞
- **主要事迹**：
  - 宁远大捷，击退努尔哈赤
  - 被崇祯帝冤杀
- **评价**：明末抗清名将

### 6. 卢象升 (1600-1639)
- **籍贯**：江苏宜兴
- **主要事迹**：
  - 抗击清军
  - 巨鹿之战殉国
- **评价**：明末忠臣

## 三、思想家

### 1. 王阳明 (1472-1529)
- **籍贯**：浙江余姚
- **身份**：心学创始人
- **主要思想**：
  - "致良知"
  - "知行合一"
  - "心即理"
- **评价**："立德、立功、立言"三不朽

### 2. 李贽 (1527-1602)
- **籍贯**：福建泉州
- **身份**：思想家
- **主要思想**：
  - 批判传统礼教
  - 反对伪道学
  - 主张个性解放
- **结局**：被视为"异端"，入狱自杀

### 3. 顾炎武 (1613-1682)
- **籍贯**：江苏昆山
- **身份**：思想家
- **主要思想**：
  - "经世致用"
  - "天下兴亡，匹夫有责"
- **评价**：清初三大儒之一

### 4. 黄宗羲 (1610-1695)
- **籍贯**：浙江余姚
- **身份**：思想家
- **主要思想**：
  - 批判君主专制
  - 提倡"天下为主，君为客"
- **评价**：中国启蒙思想家

### 5. 王夫之 (1619-1692)
- **籍贯**：湖南衡阳
- **身份**：思想家
- **主要思想**：
  - 唯物主义思想
  - "天下非一姓之私"
- **评价**：清初三大儒之一

## 四、文学家

### 1. 罗贯中 (约1330-约1400)
- **籍贯**：浙江杭州
- **代表作**：《三国演义》
- **地位**：中国四大名著之一

### 2. 施耐庵 (约1296-约1370)
- **籍贯**：江苏苏州
- **代表作**：《水浒传》
- **地位**：中国四大名著之一

### 3. 吴承恩 (约1500-约1582)
- **籍贯**：江苏淮安
- **代表作**：《西游记》
- **地位**：中国四大名著之一

### 4. 汤显祖 (1550-1616)
- **籍贯**：江西临川
- **代表作**：《牡丹亭》
- **评价**："东方的莎士比亚"

### 5. 唐伯虎 (1470-1524)
- **籍贯**：江苏苏州
- **身份**：画家、文学家
- **代表作**：绘画、诗文
- **评价**：江南四大才子之一

## 五、科学家

### 1. 李时珍 (1518-1593)
- **籍贯**：湖北蕲春
- **代表作**：《本草纲目》
- **地位**：东方药学巨典

### 2. 徐光启 (1562-1633)
- **籍贯**：上海
- **代表作**：《农政全书》
- **成就**：与利玛窦合译《几何原本》

### 3. 宋应星 (1587-约1666)
- **籍贯**：江西奉新
- **代表作**：《天工开物》
- **地位**：中国17世纪的工艺百科全书

### 4. 徐霞客 (1587-1641)
- **籍贯**：江苏江阴
- **代表作**：《徐霞客游记》
- **地位**：中国地理学的开山之作

### 5. 程大位 (1533-1606)
- **籍贯**：安徽休宁
- **代表作**：《算法统宗》
- **成就**：珠算集大成者

## 六、宦官

### 1. 郑和 (1371-1433)
- **原名**：马回回
- **身份**：航海家
- **主要事迹**：
  - 七次下西洋
  - 最远到达非洲东海岸
- **评价**：中国航海史上的伟人

### 2. 王振 (?-1449)
- **身份**：大太监
- **主要事迹**：
  - 怂恿英宗亲征瓦剌
  - 导致土木堡之变
- **评价**：明朝第一个专权的大太监

### 3. 刘瑾 (1451-1510)
- **身份**：大太监
- **主要事迹**：
  - 专权乱政
  - 贪污受贿
- **评价**："立皇帝"

### 4. 魏忠贤 (1568-1627)
- **身份**：大太监
- **主要事迹**：
  - 专权乱政
  - 残害东林党人
- **评价**："九千岁"

## 七、其他人物

### 1. 利玛窦 (1552-1610)
- **籍贯**：意大利
- **身份**：传教士
- **主要事迹**：
  - 在中国传教28年
  - 与徐光启合译《几何原本》
- **评价**：西学东渐的先驱

### 2. 徐光启 (1562-1633)
- **籍贯**：上海
- **身份**：科学家、官员
- **主要事迹**：
  - 与利玛窦合译《几何原本》
  - 编著《农政全书》
- **评价**：中国近代科学的先驱
