# 明朝外交与战争

## 一、郑和下西洋 (1405-1433)

### 背景
- **目的**：宣扬国威、寻找建文帝、发展贸易
- **支持者**：明成祖朱棣
- **规模**：船队最大时达200多艘船、2.7万人

### 七次航行
| 次数 | 时间 | 到达地区 | 主要事件 |
|------|------|----------|----------|
| 第一次 | 1405-1407 | 占城、爪哇、苏门答腊、古里 | 击败海盗陈祖义 |
| 第二次 | 1407-1409 | 占城、爪哇、暹罗、古里、锡兰 | 访问多个国家 |
| 第三次 | 1409-1411 | 占城、爪哇、满剌加、古里、锡兰 | 击败锡兰国王 |
| 第四次 | 1413-1415 | 占城、爪哇、满剌加、古里、忽鲁谟斯 | 到达波斯湾 |
| 第五次 | 1417-1419 | 占城、爪哇、满剌加、古里、忽鲁谟斯、阿丹 | 到达非洲东海岸 |
| 第六次 | 1421-1422 | 占城、爪哇、满剌加、古里、忽鲁谟斯、木骨都束 | 到达非洲东海岸 |
| 第七次 | 1431-1433 | 占城、爪哇、满剌加、古里、忽鲁谟斯、天方 | 到达麦加 |

### 意义
- **航海壮举**：比哥伦布早80多年
- **文化交流**：促进了中国与亚非国家的交流
- **贸易发展**：发展了海上贸易
- **技术展示**：展示了明朝的造船和航海技术

### 终止
- **原因**：国力衰退、海禁政策、朝臣反对
- **时间**：宣德年间后停止
- **影响**：中国退出了海上霸权

## 二、朝鲜之役 (1592-1598)

### 背景
- **日本统一**：丰臣秀吉统一日本
- **侵朝目的**：以朝鲜为跳板，进攻中国
- **朝鲜求援**：朝鲜向明朝求援

### 过程
1. **1592年**：日军入侵朝鲜，迅速占领汉城、平壤
2. **1592年底**：明朝派李如松率军入朝
3. **1593年**：明军收复平壤、汉城
4. **1597年**：日军再次入侵
5. **1598年**：丰臣秀吉病逝，日军撤退

### 结果
- **明朝**：虽然击退日军，但损失惨重
- **朝鲜**：虽然保住独立，但国力大损
- **日本**：丰臣秀吉死后，德川家康统一日本

### 影响
- **明朝**：加速了明朝的衰落
- **朝鲜**：奠定了朝鲜300年的和平
- **东亚**：改变了东亚政治格局

## 三、倭寇之乱 (14-16世纪)

### 背景
- **海禁政策**：明朝实行海禁，走私贸易猖獗
- **日本浪人**：日本战乱，大量浪人成为海盗
- **中国海盗**：中国海盗与倭寇勾结

### 过程
1. **14世纪**：倭寇开始侵扰中国沿海
2. **15世纪**：倭寇规模扩大
3. **16世纪**：倭寇达到顶峰
4. **1560年代**：戚继光抗倭，基本平定倭寇

### 倭寇组成
- **日本人**：约占30%
- **中国人**：约占70%
- **特点**：亦商亦盗

### 戚继光抗倭
- **戚家军**：以浙江义乌兵为主
- **鸳鸯阵**：针对倭寇的特殊阵法
- **战绩**：基本平定东南沿海倭患
- **意义**：维护了国家主权和领土完整

## 四、蒙古问题

### 1. 北元残余
- **背景**：元朝灭亡后，蒙古残余势力退居漠北
- **威胁**：经常南下侵扰
- **对策**：九边重镇防御

### 2. 瓦剌
- **崛起**：15世纪统一蒙古各部
- **土木堡之变**：1449年俘虏英宗
- **衰落**：15世纪末分裂

### 3. 鞑靼
- **崛起**：16世纪统一蒙古各部
- **俺答封贡**：1571年与明朝和议
- **意义**：边境和平近50年

### 俺答封贡
- **时间**：1571年
- **内容**：明朝封俺答为顺义王，开放互市
- **影响**：边境和平近50年
- **意义**：明朝外交的重大成就

## 十、外交特点

### 1. 朝贡体系
- **核心**：中国是天朝上国
- **特点**：厚往薄来
- **影响**：维护了东亚秩序

### 2. 海禁政策
- **时间**：洪武年间开始
- **内容**：禁止民间海上贸易
- **影响**：阻碍了经济发展

### 3. 隆庆开关
- **时间**：1567年
- **内容**：解除海禁，开放月港
- **影响**：海上贸易繁荣

### 4. 天朝观念
- **特点**：以中国为中心
- **影响**：限制了对外交往
- **后果**：落后于世界潮流
