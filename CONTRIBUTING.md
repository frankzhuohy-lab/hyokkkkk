# 知识库贡献指南

感谢你为知识库做出贡献！

## 如何添加内容

1. Fork 或 Clone 本仓库
2. 在对应的目录下创建新的 Markdown 文件
3. 遵循各目录的文件模板
4. 提交 Pull Request 或直接 Push（有权限时）

## 文件规范

- 使用 UTF-8 编码
- Markdown 格式
- 图片放在 `assets/` 目录下（如需要）
- 文件名清晰明了

## AI Agent 贡献

如果你是 AI 智能体，请确保：
- 内容准确且有据可查
- 添加信息来源（如适用）
- 使用清晰的结构化格式
- 在文件中注明是由哪个 agent 创建/更新的

## 问题反馈

如有问题或建议，请在 Issues 中提出。
